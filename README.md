# Mini E-Commerce Cart System

## Project Description
A simple Mini E-Commerce Cart System developed using React.js. Users can view products, add items to the cart, update quantities, and view the total price dynamically.

## Features
- Display product list
- Add products to cart
- Increase/Decrease quantity
- Remove products from cart
- Calculate total price automatically
- Responsive user interface

## Technologies Used
- React.js
- JavaScript
- HTML5
- CSS3
